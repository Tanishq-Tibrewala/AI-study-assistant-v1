# AI Study Assistant v1

AI Study Assistant v1 is a local AI-powered learning tool that helps students convert study material into notes, flashcards, quizzes, and interactive AI chat.

## Features

- Upload study material (Text or PDF)
- AI generated study notes
- Flashcards with navigation
- Quiz system with hints
- Chat with your notes
- Runs locally with Ollama

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama3
- Scikit-learn
